O que é um Array?

Um Array é uma estrutura de dados - em sua definição mais simples. Em JavaScript um array pode receber diversos valores e é tratado como um objeto, tendo seus próprios atributos e métodos.

Declarando um Array

Existem diversar formas, porém, a mais comum é:

Sintaxe:
const numeros = [20 , 130, 70, 279, 320]

Situação problema

Adicionar a um determinado array o seu valor +100

Exempo de forma Imperativa

Adicionar a um determinado array o seu valor +100

Código:

    // Solução imperativa
        const limite = numero.length -1
        let numeros100 = []

        for(let i = 100; i <= limite; i++){
            numeros100[i] = numeros[i] + 100
        }

        console.log("numeros", numeros)
        console.log("numeros +100", numeros100)

Exempo de forma Declarativa - MAP()

Problema: adicionar a um determinado array o seu valor +100

Ação: Percorre um array podendo modificar ou não. Parâmetro: calback e retorna um novo array com os valores retornados pelo callback.

Código:

Solução Declarativa
    /**
    * Parâmetros que o MAP passa
    * 
    * 1º Elemento
    * 2º Indice
    * 3º Array
    */

    const adicionar100 = (numero) => numero + 100
    const numeros100 = numeros.map(adicionar100)


    console.log("numeros", numeros)
    console.log("numeros +100", numeros100)

Situação problema

Problema: Criar um novo array com os numeros menores que 100.

Exempo de forma Imperativa

Problema: Criar um novo array com os numeros menores que 100.

Código:

Solução Imperativa
    // Problema: Criar um novo array com os numeros 
    // menores que 100.

    let menoresQue100 = []
    for(let i = 0; i<= numeros.length; i++) {
        if(numeros[i] < 100) 
            menoresQue100[i] = numeros[i]
    }

Exempo de forma Declarativa - FILTER()

Problema: Criar um novo array com os numeros menores que 100.

Código:

Solução Declarativa
    // Problema: Criar um novo array com os numeros 
    // menores que 100.

    const menorQue100 = (numero) => (numero < 100) ? true : false
    const menoresQue100 = numeros.filter(menorQue100)

Situação problema

Problema: Somar todos os números de um array.

Exempo de forma Imperativa

Problema: Somar todos os números de um array.

Código:

Solução Imperativa
    // Problema: Somar todos os numeros de um array.

    let total = 0
    for(let i = 0; i<= numeros.length; i++) {
       total+= numeros[i]
    }

Exempo de forma Declarativa - REDUCE()

Reduce: Percorre todo o array aplicando uma função (callback) e retornado um único valor e possui um Acumulador, que sempre passa a ser o elemento anterior a iteração

Problema: Somar todos os números de um array.

Parâmetros do Reduce:

  • 1º Acumulador
  • 2º Elemento (valor)
  • 3º Indice
  • 4º Array

Código:

Solução Declarativa
    // Problema: Somar todos os numeros de um array.

    const soma = (valor1, valor2) => valor1 + valor2
    const somaDeTodos = numeros.reduce(soma, 0)
